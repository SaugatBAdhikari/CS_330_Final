# CS 330 Final Project (Watering Can)

This repo contains my CS-330 OpenGL project (Visual Studio).

## Run

- Open `7-1_FinalProjectMilestones.sln` in Visual Studio
- Build + run (F5)

## Controls

- **Mouse**: look around (cursor is captured)
- **W/A/S/D**: move
- **Q/E**: down/up
- **Esc**: close window

## Notes

- Shaders are included under `shaders/` and are copied into the build output folder via a post-build step.
